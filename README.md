# Kindle-Manga-Comic-Automator-KMCA-
Uma ferramenta robusta em Python projetada para automatizar a conversão de arquivos PDF de mangás e HQs para formatos otimizados para e-readers, como o Kindle. O KMCA elimina o trabalho manual de extração, renomeação e empacotamento, entregando arquivos prontos para o Kindle Comic Converter (KCC).
